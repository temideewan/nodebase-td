# Nodebase

This application is a powerful, node-based workflow automation platform built with Next.js and TypeScript. It enables users to create, manage, and execute complex workflows with ease. The platform features a rich, interactive editor for designing workflows, robust authentication, and seamless integration with various AI providers.

### Key Technologies

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Database ORM**: [Prisma](https://www.prisma.io/)
- **Authentication**: [Polar](https://polar.sh/)
- **API**: [tRPC](https://trpc.io/)
- **Background Jobs**: [Inngest](https://www.inngest.com/)
- **AI Integration**: [Vercel AI SDK](https://sdk.vercel.ai/docs)
- **Linting and Formatting**: [Biome](https://biomejs.dev/)
- **Error Monitoring**: [Sentry](https://sentry.io/)

### Getting Started

To get started with the development environment, follow these steps:

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Run the development server**:
   ```bash
   npm run dev:all
   ```
This command starts the Next.js development server and the Inngest development server concurrently.

3. **Open the application**:
   Open [http://localhost:3000](http://localhost:3000) in your browser to see the application in action.
